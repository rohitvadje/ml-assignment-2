Problem: Breast cancer classification.

Dataset: sklearn breast cancer dataset (569 rows, 30 features).

Models:

Decision Tree
KNN
Naive Bayes
Random Forest
XGBoost

Metrics: Accuracy, AUC, Precision, Recall, F1, MCC.

Observation:
Random Forest and XGBoost performed best.
